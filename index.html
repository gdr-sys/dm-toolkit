    ☁️ Configura Google Drive

    Per sincronizzare e collaborare tramite Google Drive ti serve un Client ID OAuth 2.0 gratuito.

    1. Vai su console.cloud.google.com
    2. Crea progetto → abilita Google Drive API
    3. Credenziali → OAuth 2.0 Client ID → tipo Web
    4. Aggiungi https://gdr-sys.github.io come origine autorizzata
    5. Copia il Client ID qui sotto

    Il Client ID viene salvato solo sul tuo dispositivo.

    🔥 Configura Firebase

    Firebase è gratuito e permette collaborazione in tempo reale. Setup in 5 minuti:

    1. Vai su console.firebase.google.com
    2. Crea progetto → abilita Google Authentication
    3. Crea Realtime Database (modalità test)
    4. Impostazioni progetto → Aggiungi app Web → copia la config
    5. Aggiungi gdr-sys.github.io ai domini autorizzati
    Incolla qui la tua Firebase config (oggetto JS):

    La config viene salvata nel tuo browser. Non contiene dati sensibili — è la configurazione pubblica dell'app.
